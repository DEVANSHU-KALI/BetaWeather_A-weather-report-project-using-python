# what is this project
- This a weather report project built using the python and uses openweather api to get the weather details.

# Features
- Temperature
- Wind speed
- Wind speed
- TIme of day
- Uses the OpenWeatherMap API
- Includes a search function that allows users to search for weather information for other cities.
- Includes a dark mode version in a folder of it's own

# How-To-Use
- clone this repo from the git page.
- install the requirements, if you get error regarding the flask version just upgrade it to the latest version.
